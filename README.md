# IntakePad

Messy notes in. Clean GitHub issue out.

Turn messy notes, Slack threads, bug reports, and rough ideas into clean GitHub issues — fast.

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

Stack: Next.js 16, TypeScript, Tailwind CSS.

## Milestones

- **M0**: Clickable local prototype — layout, mock templates, editor, autosave, copy Markdown
- **M1**: GitHub templates — real template loading from `.github/ISSUE_TEMPLATE/`
- **M2**: Issue creation — create real GitHub issues
- **M3**: AI enhancement — optional AI normalization
- **M4**: Hardening — errors, rate limits, tests, polish
